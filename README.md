# rulai-wget


mwget 使用说明

```
这是把mwget 改为 python3 的版本, 用来做一些大文件的下载

optional arguments:
-V, --version         输出版本
-d, --debug           默认为False
--file_name FILE_NAME
下载后的文件名
--retry_times RETRY_TIMES
重试的次数
--complicate_num COMPLICATE_NUM
多进程的数量
--chunk_size CHUNK_SIZE
分片的文件大小,默认256,随网速而定
--referer REFERER     referer 请求的referer
-h, --help            获取帮助文档
```
